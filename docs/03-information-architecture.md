Version: 1.0

Status: Approved

# Purpose

This document defines the information architecture of the Mato Cashew Platform.

It serves as the blueprint for:

- Navigation
- URL Structure
- Content Organization
- SEO
- Breadcrumbs
- Sitemap
- Search
- Internal Linking
- Future CMS Integration

No UI implementation should begin before the Information Architecture is approved.

---

# Platform Vision

Mato Cashew is not only a product website.

It is a digital platform designed to become the most trusted source of Cambodian cashew information for both domestic and international audiences.

Primary audiences include:

- Consumers
- Retail Customers
- Wholesale Buyers
- Importers
- Exporters
- Food Manufacturers
- Researchers
- Investors
- Students
- Media

# Information Architecture Principles

Single Source of Truth

Every page belongs to one primary category.

Navigation data should not be duplicated.

Breadcrumbs should be generated automatically.

URLs must remain stable.

Content should always be internally connected.

# Success Criteria

Users can find information within three clicks.

Navigation remains consistent across all pages.

Every page has a logical parent.

Breadcrumbs are generated automatically.

Search indexes every content type.

The architecture supports future CMS integration without redesign.
