# Purpose

This document defines the content strategy for the Mato Cashew Platform.

It establishes:

- Content types
- Editorial standards
- SEO strategy
- Internal linking
- Multilingual workflow
- Publishing workflow

This document ensures that all future content follows a consistent structure.

---

# Vision

Mato Cashew is not only selling cashew products.

Our goal is to become the leading Cambodian Cashew Knowledge Platform.

Content should educate before it sells.

Trust comes before conversion.
